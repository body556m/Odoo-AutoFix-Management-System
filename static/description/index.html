AutoFix - Car Workshop Management System

A comprehensive Odoo 17 module for managing car workshop operations

Key Features

🚗 Vehicle Management

  • Register vehicles with brand, model, year, VIN, mileage, fuel type
  • Master data for car brands and models
  • Complete service history tracking per vehicle

🔧 Service Reception

  • Create and manage service orders with customer complaints
  • Predefined service types: Mechanical, Electrical, Body Work, Paint, A/C, Tires
  • Workflow: Draft → In Progress → Done → Cancelled

📋 Work Orders

  • Assign mechanics to specific tasks
  • Track parts used with stock integration
  • Quality control and time tracking

📅 Appointments

  • Schedule customer visits with date/time slots
  • Visual calendar for appointment management
  • Auto-create service reception on arrival

🔍 Vehicle Inspections

  • Create reusable checklist templates
  • Record detailed inspection results with conditions
  • Categories: Engine, Brakes, Suspension, Electrical, Body, Tires

🛡️ Warranty Tracking

  • Track warranty for parts, labor, and full coverage
  • Manage warranty claims with approval workflow
  • Auto-expiry via scheduled actions

⭐ Customer Feedback

  • 1-5 star rating system
  • Track service quality, cleanliness, timeliness
  • Recommendation tracking

💰 Financial Management

  • Petty cash expense tracking with approvals
  • Employee payroll management
  • Invoice generation from service receptions

Security Groups

Group Description
AutoFix / Manager Full access to all features
AutoFix / Accountant Access to petty cash and payroll
AutoFix / Receptionist Manage receptions and appointments
AutoFix / Mechanic Manage own work orders

REST API Endpoints

Endpoint Method Description
/api/autofix/receptions GET List service receptions
/api/autofix/work-orders GET List work orders
/api/autofix/cars GET List registered cars
/api/autofix/dashboard GET Get dashboard KPIs
